## Objetivos de Aprendizagem

Ao final desta matéria, o aluno deverá ser capaz de:

<Steps>

1. Diferenciar dados estruturados de não estruturados.

2. Entender o modelo relacional (Tabelas, Chaves Primárias e Chaves Estrangeiras).

3. Escrever consultas SQL simples (`SELECT`, `FROM`, `WHERE`) para filtrar dados.

4. Compreender a importância da integridade dos dados para a previsão.

</Steps>

## Tópicos Abordados

-   **Estrutura de Dados**: O que é um SGDB e por que não usar apenas Excel.
-   **Modelagem Básica**: Como as informações se conectam (ex: Tabela de Pedidos x Tabela de Fornecedores).
-   **SQL para Analistas**: Extração de dados (Queries básicas).
-   **Qualidade de Dados**: Garbage In, Garbage Out.
