## Objetivos de Aprendizagem

Ao final desta matéria, o aluno deverá ser capaz de:
1.  Utilizar funções lógicas (`SE`, `E`, `OU`) para tratamento de dados.
2.  Realizar buscas de dados eficientes com `PROCV`, `ÍNDICE` e `CORRESP` (e `XLOOKUP`).
3.  Criar Tabelas Dinâmicas (Pivot Tables) para resumir grandes volumes de dados.
4.  Construir gráficos claros para apresentação de resultados.

## Tópicos Abordados

-   **Tratamento de Dados**: Limpeza de texto, datas e formatação condicional.
-   **Fórmulas Essenciais**: Soma, Contagem, Médias condicionais (`SOMASE`, `CONT.SE`).
-   **Busca e Referência**: Cruzando tabelas diferentes.
-   **Análise Dinâmica**: Pivot Tables e Pivot Charts.
